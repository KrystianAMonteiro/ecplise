Comandos para subir arquivos no github;
abrir o cmd na pasta desejada e colocar os comandos em ordem;
- rd .git /S/Q
- git init
- git add .
- git commit -m "first commit"
- git remote add origin <link do repositorio>
- git push -u origin master
